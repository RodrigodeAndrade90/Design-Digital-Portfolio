ANÁLISE ESPAÇO TEMPORAL DE TURBIDEZ DA BACIA DO RIO PARAÍBA DO SUL DE 2013 a 2017

Trabalho acadêmico de graduação apresentado como atividade de conclusão do curso de Tecnólogo em Geoprocessamento, pela FATEC de Jacareí, cujo nesse trabalho acadêmico foi uma análise espaço temporal de turbidez da bacia do Rio Paraíba do Sul entre 2013 a 2017.

O objetivo do trabalho de graduação é efetuar uma análise espaço temporal ocorridos neste determinado período, analisando o indicador turbidez para a qualidade da água da bacia, constatar se as variáveis levantadas atendem a qualidade das águas da bacia do rio Paraíba do Sul, conforme os critérios regidos pelas legislações e normas vigentes, examinar a variabilidade espacial da qualidade das águas mediante do Índice da Qualidade das Águas (IQA), e confrontar os dados levantados dos indicadores, em conformidade da qualidade das águas da bacia do rio Paraíba do Sul, empregando métodos diversificados de pesquisa.

As ferramentas utilizadas pelo trabalho acadêmico foram feitos pelo software QGis. Com os dados adquiridos pela Cetesb, e com a interpolação das imagens para dos meses analisados dos anos de 2013 a 2017 mostra os valores atribuídos para análise na qual o padrão de referência da CONAMA (até 100 UNT).

IMPLEMENTAÇÃO DE TERMINAL REMOTO ATRAVÉS DE SERVIÇOS EM NUVEM

Trabalho acadêmico apresentado como atividade de conclusão do curso de Tecno em Redes de Computadores, pela ETEC de Jacareí, cujo nesse trabalho acadêmico foi uma implementação de terminal remoto através de serviços em nuvem.

O objetivo do trabalho acadêmico foi implementar um servidor virtualizado em nuvem para disponibilizar uma área de trabalho com aplicativos, configurado com login e senha, através das ferramentas Microsoft Windows Server, a plataforma da Amazon Web Services (AWS), e o computador de placa, o Raspberry Pi 3

Com essa implementação, os recursos de virtualização foram utilizados pela linguagem de programação Java pelo software IDE Lazarus, e com as atribuições nas interfaces gráficas e com as colocações de recursos de aplicações na área de trabalho do usuário no seu Raspberry. Com os arquivos colocados, o Raspberry funciona como software, mas com a virtualização de nuvem privada, e os seus usuários são colocados para sua entrada através do Windows Server 2016, inteligados o seu banco de dados para AWS.
